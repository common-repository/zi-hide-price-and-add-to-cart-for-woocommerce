== Description ==

This WP plugin hides <strong>price</strong> and <strong>add to cart button</strong> for WooCommerce products under a specific category (or specific categories ) which you can choose.

When you add a WC product category, you'll have an NEW option to hide the prices and add to cart buttons for all products under that category. This happens on single product pages and on archives and shop pages.

To have a clear situation about the product categories which have this option enabled, there is a new column added in categories table.

This plugin works with WooCommerce 3.6.0 or higher. It works with <strong>WooCommerce 8.5.1</strong>.

If you need more info please check the plugin on <a href="https://www.zeninvader.com/hide-price-add-to-cart-woocommerce/">Zen Invader</a> website.


<a href="https://www.zeninvader.com/hide-price-add-to-cart-woocommerce/">Go to plugin's page</a> if you have other questions or suggestions.


== Installation ==


1. Upload `zi-hide-price-and-add-to-cart-for-woocommerce.1.5.0.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to a WooCommerce product category and you will notice the new option: "Hide Price and Add to Cart button for this category".
